The Overview

Whether bright or sparks, it is up to you to decide based on your taste. Both ConEmu and Windows Terminal are shells operating on top of PowerShell.
Here is what we will do,

  1. Setup a Terminal/Shell
  2. Configure Powershell: Colors + Fonts (PowerLine)
  3. Add Splash ASCII art
  4. Add some alias
Setup the terminal

I am using PowerShell’s latest release-rc-2 and using chocolatey to manage all my package.

choco install microsoft-windows-terminal
choco install cascadiacodepl

The PowerShell setup itself we can refer to Windows Terminal Setup â€” PowerLine-Setup1

However, on top of that, I still need to register the trusted source for a better setup experience.

  • Examine our PowerShell repository 
    Get-PSRepository
  • Set the PSGallery as a trusted source 
    Set-PSRepository -Name 'PSGallery' -InstallationPolicy Trusted
  • Now we’re ready to install the PowerShell modules: posh-git, on-my-post, and PSReadLine 
    Install-Module posh-git -Scope CurrentUser
Install-Module oh-my-posh -Scope CurrentUser
Install-Module -Name PSReadLine -Scope CurrentUser -Force -SkipPublisherCheck
  • To prettify it more, we another module, Get-ChildItemColor. 
    Install-Module -Name Get-ChildItemColor

Once those modules are installed, we need to enable them in our PowerShell profile, through notepad.

notepad $Profile

Or, if you are a fan of notepad++ like me, we can make an alias for it,

Set-Alias -Name npp -Value 'C:\Program Files\Notepad++\notepad++.exe' -Description "Alias to Notepad++"

with that, you can open the $Profile using npp, a notepad++’s alias.

npp $Profile
PowerShell & Windows Terminal setup
  1. In PowerShell’s $Profile, we will have the following imported modules and theme,
Import-Module posh-git
Import-Module oh-my-posh
Import-Module Get-ChildItemColor
Set-Theme Paradox

Set-Alias -Name npp -Value 'C:\Program Files\Notepad++\notepad++.exe' -Description "Alias to Notepad++"
  1. In Windows Terminal, we add the installed Cascadia Code PL along with the font size, depends on our screen resolution.
    For the complete settings list, we can refer it to a Windows Terminal profile setting.
"fontFace": "Cascadia Code PL",
"cursorShape": "filledBox",

There are plenty of options like a background picture, to that extend, I leave it to the creative thought out there. For me, these 2 options are enough. Light and minimal.
