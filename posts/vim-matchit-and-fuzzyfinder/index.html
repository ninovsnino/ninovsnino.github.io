Vim MatchIt and FuzzyFinder

May 29, 2016 One-minute read
Software • vim

Here is my addition to my .vimrc config file this week. https://gist.github.com/ninovsnino/f55aadeb8c2e481d6bbf5d9a42152c71

The First part is to load matchit.vim that already part of Vim distribution, it aiding a great help for me to browse through html file, it extend the % vim command.
The second part is a bit tricky for FuzzyFinder plugin user, I want to exclude directory, in this case is “venv” directory, that apparently the regex matching work well with fuf_file_exclude instead of fuf_dir_exclude.

You can add you excluded list by extending and following the line #9 from snippet above.
